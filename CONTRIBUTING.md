# Contributing to Mohamed Tayel Portfolio

Thank you for your interest in contributing! This portfolio is mainly a personal showcase, but constructive feedback, bug reports, and minor improvements are always welcome.

## How to Contribute

1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes following Conventional Commits (`git commit -m 'feat: add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Code of Conduct

Please ensure your interactions are respectful and constructive.

## Reporting Bugs

If you find a bug, please open an issue with:
- A clear description of the problem
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)
